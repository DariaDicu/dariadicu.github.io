⁣ ⁣ Bayesian Inference for
Absolute Quantitation of Mitochondrial DNA

⁣


Quantitation of mitochondrial DNA (mtDNA) in single cells is key in improving the estimation of heteroplasmy, which would further aid the understanding of mechanisms underlying mtDNA mutations. Current methods are limited; they do not take into account the stochasiticity of the qPCR amplification process and do not provide an error measurement. This project looks at adapting the existing research on qPCR quantitation using Hidden Markov Models to mtDNA experimental data. It tackles particularities that are not discussed in literature, and provides a model that can be extended in the future by adding different error sources.

Experimental data

The experimental protocol for measuring the amounts of mutant (m) and total (N) mtDNA is based on quantitative PCR. This is a technique for amplifying a known segment from an initial set of DNA molecules (more details in Background).

For a single cell, the contents are split into two wells. The molecules on each well are amplified using qPCR in order to obtain two quantities: m/2 from the first well and N/2 from the other.

The resulting data consists of amplification curves for each of the two wells. For each consecutive cycle, the experiment yields a fluorescence read that is proportional to the number of mtDNA molecules at that cycle.

We use the amplification curve data to estimate and model the measurement error in the initial number of molecules m/2 and N/2. One key aspect is correcly relating the fluorescence read value with the actual number of molecules.

Aims

The aim of this project is to model the error in mtDNA quantification for single-cell data. A common method for absolute quantification of mtDNA from qPCR experiments is the standard curve method (more details in Background). The amplification process is modelled as a Hidden Markov Model, and Bayesian inference (pseudo-marginal Metropolis Hastings) is used to infer the parameters of this model that best fit the amplification curve.
