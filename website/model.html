Modelling the uncertainty and stochasticity

We wish to obtain a mathematical model for the qPCR experiment that captures both the stochasticity of the amplification at each cycle and the uncertainty in fluorescence reads.

For a qPCR experiment with n cycles, our mathematical model has the following random variables:

  • X0, the initial number of molecules
  • X1, …, Xn, the number of molecules present after each cycle
  • F1, …, Fn, the fluorescence reads observed after each cycle

The aim is to relate these random variables through a model. Then, using an inference method (see Inference)), we can estimate the marginal probability P(X0 = x0 | f1, …, fn) of the initial number of molecules being some constant x0 given a set of known observations f1, …, fn. The inference method additionally estimates other probabilities that may be useful in understanding the results (see Results)).

We view the qPCR experiment as a random branching process with an amplification efficiency r that is constant during the exponential phase. The efficiency indicates how likely it is for each individual molecule to be amplified. A Binomial distribution is used to capture the number of new copies at each cycle, as each molecule undergoes an independent Bernoulli trial with probability r. The following equation relates the number of molecules from consecutive cycles: X(i+1) = X(i) + Binom(X(i), r), where i >= 0

The fluorescence intensity emitted by the reporter dye at a given cycle i is proportional to the number of molecules Xi. However, the measurements are made in the presence of background fluorescence, which we model as an additive Gaussian noise. The data we use is background corrected, so the Gaussian distribution has mean 0 and unknown variance s2: F(i) = a*X(i) + N(0, s2).

Hidden Markov Model

These relationships can be integrated in a Hidden Markov Model, where the fluorescence reads are observed states and X1, …, Xn are hidden states. The parameters of the model are the initial number of molecules X0, the efficiency r and the noise s2.
